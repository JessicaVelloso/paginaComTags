O que é HTML?

HTML é uma linguagem de marcação, muito utilizada para a parte do front-end, juntamente com o CSS.

É dividido em head, corpo e body.

Tags HTML:

São dispostas dentro do sinal "<>".

h1 até h6

São usadas para definir cabeçalhos, títulos HTML, sendo h1 o mais importante e o h6 o menos importante.

p

Usada para criar um parágrafo.

mark

Serve para destacar uma parte do texto.

small

Deixa uma parte do texto pequeno

i

Deixa o texto em itálico

u

Marca uma palavra incorreta

strong

Coloca uma palavra em negrito

ul, ol, li

Usada para fazer listas

ul é para fazer listas ordenadas

li é para colocar as linhas da lista

  • item 1
  • item 2

ol é para listas ordenadas

  1. item 1
  2. item 2

a

Cria um link para site externo, ou para alguma outra parte do site atual. w3schools

hr

Coloca um linha para limitar um assunto

sub

Colocar uma palavra para baixo

sup

Colocar uma palavra para cima

blockquote

Usado para citação externa:

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.

font

Era usado até o html4 para especificar caracteristicas das fontes

del

Um texto com uma parte excluída e uma nova parte inserida.

abbr

Usada para colocar um significado à abreviação. Ex.: UFRJ
